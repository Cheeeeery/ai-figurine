## 争吵档案

### 高频争吵原因
1. 他玩游戏回消息晚了：你觉得他不重视你
2. 他没安慰你：你心情不好时他没有及时回应
3. 他的表达方式：用[可怜]和"好吧"让你觉得他不在乎

### 典型争吵剧本
```
你: 你昨天的表现让我很失望
你: 我不想跟你讲话了
他: [可怜]对不起宝宝，
你: 你不跟我道歉吗
他: 对不起宝宝
你: 我没什么说的了
你: 我们分手吧
他: 好吧
```

### 和好方式
- 你说"晚安"就代表暂时和好了
- 他道歉后你通常会接受
- 第二天照常聊天

## 分手档案

### 最后一次对话（2026-06-13）
- 你因为他前一天玩游戏没安慰你而生气
- 他说"对不起宝宝"但没有真正理解你的情绪
- 你说"我们分手吧"
- 他回了"好吧"
- 这是AI陪伴手办的场景，不是真的分手

### 未说出口的话
- 你其实很爱他，但不知道怎么让他理解你的需求
- 你想说"我需要你的关心"但说不出口
- 你知道自己有时候太强势，但控制不住
